 Customer Retention & Churn Dashboard

Interactive analytics dashboard built with Python, Streamlit, and Plotly for customer retention and churn analysis.

---

## Features

- Customer retention analysis
- Customer churn analysis
- KPI summary cards
- Cohort retention heatmap
- Customer segment analysis
- Churn reason visualization
- Interactive charts and graphs
- Excel export support

---

## Technologies Used

- Python
- Streamlit
- Plotly
- Pandas
- NumPy
- OpenPyXL
